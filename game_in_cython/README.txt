RUNNING THE "POST-APOCALYPTIC FANTASY" GAME

1. Install Cython: sudo pip install Cython
2. Navigate to the game_in_cython directory
3. Convert the shell script to executable: chmod u+x install.sh
4. Run the script: ./install.sh
5. After it finished, navigate to bin: cd bin
6. Start the Python interpreter: python
7. Type in the following: import test_act_loader
8. You can use "exit()" or Ctrl+D to close the Python interpreter

Easy.
